#LireXML
Ce dossier contient du code source qui permet de lire le fichier xml de freeplane afin d'en extraire les données pertinante.

Cependant ce répertoire n'est pas indispensable, car il nous a servis juste pour la phase de développement où il fallait bien séparé les codes des différentes équipes. En effet toutes les classes présentes dans ce dossier ont été gréffé au projet principale dans le répertoire "ProjetM2".
