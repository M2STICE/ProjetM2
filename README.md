# ProjetM2
Le projet de M2 Informatique 2015-2016 de l'université des antilles

Le projet est composé de trois répertoires:
 - LireXML: Extration des données du fichier XML et insertion dans la base de données.
 - ProjetM2: C'est le projet principale, il contient toute l'application.
 - SQLCreator: Contient les fichiers SQL de création de la base de données, des tables et des utilisateurs ainsi que l'attribution des droits et les requêtes à prévoire pour l'utilisation de l'application.
 
La création de ces trois dossiers nous permet de bien séparé le travail de chaque équipe.
